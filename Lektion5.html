Lektion 5: HTTP und Adressen

  • Home

Zusammenfassung

HTTP (Hypertext Transfer Protocol) ist das Protokoll, das für die Übertragung von Webseiten verwendet wird. URLs (Uniform Resource Locators) spezifizieren Adressen von Ressourcen im Web.

Code-Beispiel


GET /index.html HTTP/1.1
Host: www.example.com

HTTP/1.1 200 OK
Content-Type: text/html

<html>
<body>
  <h1>Hallo Welt</h1>
</body>
</html>
